### Project Overview

A modern full-stack online store for clothing with a user-friendly UI/UX. Complete e-commerce solution with product management, authentication, and an admin dashboard.

### Tech Stack

Next.js with TypeScript for the full-stack implementation. Tailwind CSS with DaisyUI for rapid UI development. Prisma ORM with MongoDB for data persistence. Firebase Storage for image management.

### Key Features

- Home page, category and product listing pages
- Individual product pages with image gallery
- Authentication flow (sign up, sign in, OAuth)
- Admin dashboard for product and order management
- Shopping cart and checkout flow
- Firebase image storage integration
