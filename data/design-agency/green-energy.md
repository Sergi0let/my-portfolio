### Project Overview

A multi-page website for a green energy company with an integrated blog powered by Sanity.io as a headless CMS. Features animated sections and a fully manageable content structure.

### Tech Stack

Next.js with TypeScript for type-safe server-side rendering and static generation. Sanity.io as the headless CMS for content management. GSAP for scroll animations. Tailwind CSS for utility-first styling.

### Key Features

- Multi-page structure with dynamic routing
- Headless CMS with Sanity.io Studio
- Blog with dynamic content from Sanity
- Scroll-based GSAP animations
- TypeScript throughout for type safety
- ISR (Incremental Static Regeneration) for blog pages
