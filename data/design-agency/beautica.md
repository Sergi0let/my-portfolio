### Project Overview

Beautica is a landing page built for a licensed cosmetologist based in Kyiv. The client needed a professional online presence to showcase her services, build trust with potential clients, and drive appointment bookings.

The site covers the full customer journey — from discovering the specialist and browsing the services catalog to reading real client reviews and submitting a booking request directly through the contact form.

### Tech Stack

Built with Next.js 14 and TypeScript for type-safe server-side rendering and fast page loads. Tailwind CSS handles all styling with a custom brand palette, while shadcn/ui components ensure accessible and consistent UI patterns. Images are optimized using Next.js built-in tooling.

### Key Features

- Services catalog with 5 treatment types and transparent pricing
- Smooth scroll navigation between sections
- "Before & After" interactive gallery with category filters
- Client testimonials slider
- Partner brands section
- Accordion FAQ block
- Contact form with phone mask input and validation
- Fully responsive across all screen sizes
- SEO-optimized with semantic HTML, meta tags, and structured headings
